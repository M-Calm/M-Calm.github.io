个人博客创建流程

1. 创建Github 域名和空间

1.1注册

首先你需要注册一个Github账号，已有的可以下翻去1.2 创建仓库，注意username，这会影响到你的域名，你的域名将会是 username.github.io ，所以认真的取个名字吧。

​ 注册过程可能需要验证你的邮箱，其他就不在赘述。

1.2 创建仓库

然后需要创建一个仓库(repository) 来存储我们的网站，点击首页任意位置出现的 New repository按钮创建仓库, Respository name 中的username.github.io 的username 一定与前面的Owner 一致，记住你的username下面会用到。

​ 第一步就已经完成了，下面是安装。

2. 安装

Hexo 可以说是目前最流行的博客框架了，基于Nodejs，更多信息可以google，下面需要安装的工具包括 Git，Nodejs，Hexo。（Windows 用户自行搜索这些工具，直接安装即可，试过基本没啥问题）

  1. 安装Git

    这是windows版本下载地址

    https://git-scm.com/download/win

  1. 安装Hexo
    以上所有都安装完成之后再安装Hexo
    $ sudo npm install hexo-cli -g
    所有必须工具已经安装完成，下面我们就可以生成博客，上传至我们的Github 仓库了。

编写，发布

接下来我们需要用Hexo初始化一个博客，然后更改一些自定义的配置，或者加上自己喜欢的主题，写上第一篇文章，然后发布到自己的个人Github网站(username.github.io)。

  1. 创建博客
    将下面的 username 替换成你自己的username(其实也无所谓，作者强迫症)，执行成功后，会创建出一个名为 username.github.io 的文件夹。
    $ hexo init username.github.io

  2. 更改配置
    主题安装
    为了使博客不太难看，我们需要安装一个主题，切换至刚刚生成的Hexo 目录，安装主题
    $ cd username.github.io
    $ git clone https://github.com/iissnan/hexo-theme-next themes/next
    这里选了一个极简的主题，也是Hexo众多主题中最受欢迎的一个。上面出现的喵神的主题 在这里。Hexo也有更多主题供你选择。
    基础配置：打开文件位置username.github.io/_config.yml修改几个键值对，下面把几个必须设置的列出来按需求修改，记得保存， 还有注意配置的键值之间一定要有空格。更多设置…
    title: dimsky 的 9 维空间 //你博客的名字
    author: dimsky //你的名字
    language: zh-Hans //语言 中文
    theme: next //刚刚安装的主题名称
    deploy:
    type: git //使用Git 发布
    repo: https://github.com/username/username.github.io.git // 刚创建的Github仓库
    主题配置：
    主题配置文件在username.github.io/themes/next/_config.yml中修改，这里略过。设置详情

  3. 写文章
    所有基础框架都已经创建完成，接下来可以开始写你的第一篇博客了
    在username.github.io/source/_posts下创建你的第一个博客吧，例如，创建一个名为FirstNight.md的文件，用Markdown大肆发挥吧，注意保存。
    如：

  1. 测试
    $ hexo s
    测试服务启动，你可以在浏览器中输入https://localhost:4000 访问了。

  2. 安装hexo-deployer-git自动部署发布工具
    $ npm install hexo-deployer-git –save

  3. 发布
    测试没问题后，我们就生成静态网页文件发布至我们的Github pages 中。
    $ hexo clean && hexo g && hexo d
    如果这是你的第一次，终端会让你输入Github 的邮箱和密码，正确输入后，骚等片刻，就会把你的博客上传至Github 了。以后在每次把博客写完后，执行一下这个命令就可以直接发布了，灰常苏胡。

  4. 5分钟应该快到了
    是不是很快，恭喜你能走到这一步，你的博客已经完成了，在浏览器中输入 http://dimsky.github.io 就能够访问了。

是谁叩开了我的四月轩窗

​ 四月，院落里的春花泛起一缕缕的清香，那一缕缕的清香在夜风寂静的时候，惊乱了心底的那片湖。倚窗望向星空，看时光缠绕着指尖，心湖的深处荡起了岁月醇厚的歌。一丝情愫，随着杨柳的树梢，在轻风弹奏的声音中，不着痕迹地缓缓滑过。喜欢静静的看着星空遐思，感觉这样能够温暖心灵，浅品流年清欢。寂寞的灵魂披着孤独的寒衣，越过沟沟壑壑，走过山河岚烟，安静地在字里行间中学会照顾自己坚强地活着。

​ 四月，是情愫容易随波逐浪的季节，每一秒、每一分，都在周而复始地增厚年轮。时常想，这种素静安逸的日子否就是对于岁月的虚度。只是，当年华随着岁月逐渐老去时，才明白，还是情愿做本真的自己。隔着岚烟水色，看着月色下都市灯火辉煌的街道，心湖的波澜起伏不定。不眠的夜里，悄悄与星星对话，和弯月谈心，尽可能让孤单的自己在忧愁中释然，安安静静地做自己想做的事情，自己赋予自己最美的时光。

​ 四月，也是情动的季节，只要是正常的人，荷尔蒙刺激下的冲动，会使人精神异常激昂，做出一些随自己情绪波动的事情。比如爱情，这个季节对于爱的表白更能让对方所接受。一段让人眷念的青梅往事，有时候不需要任何理由，就会一段段从脑洞的深处浮现出来，不需要有前因，更无关风月，只是深深地爱了。岁月悠悠，当窗外的城市的每个角落花开灿灿，我倚在窗旁墨香素笺上，寻觅着谁能陪我在这烂漫的日子里安度流年。

​ 四月，夜色融融，呼吸着院落中春花的幽香，轻衣漫步月色下，往事犹如眼前的点点繁星，忽隐忽现。热恋时的山盟海誓，仿佛犹在耳边，叹！叹！叹！人生若只如初见，何事秋风悲画扇。姹紫嫣红的开始，寒雪梅香凄情的结尾。惜！惜！惜！惜时光的日记，写上情深意重的永远，不想再有世态炎凉的泪意和惘然！虽然世间事都躲不过因果的无奈，我依然深信，在每个浪花起落间，都有不曾遗失的梦，寻觅初心时那承诺过的永恒红尘往事。( 文章阅读网：www.sanwen.net )

​ 四月，春渐深，花开花又凋谢，一些人从眼前走了，缘分便尽了，一些人出现在眼前，另一份缘分闯入你的生活。都说四月春雨缠绵，就连安居在素笺上的文字，也被春雨浸润得湿淋淋的，写之又读之，不知道是文字的美，还是词句的华丽，使人心生眷恋！都说，春花绚丽的色彩，会吸引充满希冀的眼眸，让我们在春天的路上奋力往前行走。独自沉溺于一片寂静中，品味着过往，静静地沉思，总是把满满的困惑和伤感藏匿于心中。

​ 四月，是充满诗意的季节，也是充满期待和希望的季节。然而，春的风情，终究承载不了永恒故事，它只会播种，将美好的念想根植于心的深处生根发芽。月色洒落下的院落中，盈盈的绿意里，绽开着的花枝俏蕊随风摇曳，婆娑起舞。风儿敲打着展开的窗棂，书架旁边悬挂的那串用子弹壳做的风铃叮叮当当奏起悦耳的声音，花香溢满清室，一杯香茗，安度静好时光。

​ 四月，清晨的阳光从轩窗的缝隙中扑了进来 ，暖融融的。拉开门户，穿行在阳光明媚的春光里，让花香绕肩，任春花飞舞，在这个充满诗意香浓的季节里，静静地领悟着人生的真谛，人生，就是一路行走，一路感悟，无论经历多少的风雨，多少的坎坷，请记得一定要带上自己的微笑上路，让自己和身边的人都能有一个好心情，彼此互相包容，互相照顾，纵是心绪不小心触摸了忧伤，相信，那浓郁甜美的花蕊里，一定会释放一缕缕清香，来抚平生活中的伤疼和疤痕……
